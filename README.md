# Android Python Editor

一个在Android设备上运行的Python代码编辑器，支持代码高亮、自动补全、包管理和Git功能。

## 功能特点

- 🎨 Python代码高亮显示
  - 关键字高亮
  - 内置函数高亮
  - 字符串高亮
  - 数字高亮
  - 注释高亮
  - 行号显示

- 📝 代码编辑功能
  - 智能代码补全
  - 实时语法高亮
  - 清空代码功能

- 📦 包管理功能
  - 支持pip安装第三方包
  - 内置gitpython支持

- 🔄 版本更新
  - 自动检查更新
  - 支持强制更新
  - 更新提醒对话框

## 技术栈

- Kotlin
- Python (Chaquopy)
- Android Jetpack
- CodeView
- JGit

## 系统要求

- Android 7.0 (API 24) 或更高版本
- 设备需要联网功能
- 需要存储权限（用于保存Python包）

## 安装说明

1. 从Release页面下载最新APK
2. 允许安装未知来源的应用
3. 安装完成后授予存储权限

## 使用方法

1. 在代码编辑区输入Python代码
2. 点击"运行代码"执行
3. 在输出区查看结果
4. 需要安装包时，在输入框填写包名，点击"安装包"
